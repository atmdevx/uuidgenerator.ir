# UUID Generator 🔐

> Professional UUID Generator for developers

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)
[![Live Demo](https://img.shields.io/badge/demo-online-green.svg)](https://uuidgenerator.ir)

## ✨ Features

- ✅ **UUID v1** - Time-based + MAC address
- ✅ **UUID v4** - Cryptographically secure random
- ✅ **UUID v7** - Time-ordered + random (sortable)
- ✅ **Batch Generation** - Generate up to 100 UUIDs
- ✅ **Copy to Clipboard** - One-click copy
- ✅ **Download as TXT** - Export to file
- ✅ **REST API** - Developer-friendly endpoints
- ✅ **Dark Theme** - Premium zinc-950 design
- ✅ **Privacy First** - No data collection

## 🚀 Quick Start

```bash
git clone https://github.com/atmdevx/uuidgenerator.ir.git
cd uuidgenerator.ir
open index.html
```
